# 🐱 Purrfect Break

A Chrome extension I built for fun — a cat shows up every 15 minutes to remind you to take a break, drink water, and stretch.

## What it does

- A cat walks onto your screen every 15 minutes
- Shows a rotating mindfulness tip (grounding, breathing, affirmations, body scan, etc.)
- Has an animated SVG chair stretch guide that cycles through 5 different exercise sets each break
- Tracks your break streak and cat happiness
- Guided breathing ring synced to a 4-7-8 pattern
- 3 unlockable cat personalities (Mochi, Biscuit, Noodle) with 150+ unique cat puns
- Built with Chrome MV3, vanilla JS, no dependencies

## Install locally

1. Clone this repo
2. Go to `chrome://extensions`
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** and select the folder
5. Click the cat icon in your toolbar

## Stack

- Chrome Extension Manifest V3
- Vanilla JS / CSS — zero dependencies
- Cat GIFs from [CATAAS](https://cataas.com)
